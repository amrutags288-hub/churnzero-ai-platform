# 🚀 AI-Powered Banking Churn Prediction System

An AI-driven Banking Customer Churn Prediction and Retention Intelligence Platform built using **Machine Learning, FastAPI, and React**.

This project predicts whether a banking customer is likely to leave the bank and provides intelligent retention recommendations.

---

# 📌 Features

- ✅ Customer churn prediction using Machine Learning
- ✅ Risk level analysis (Low / Medium / High)
- ✅ Churn probability calculation
- ✅ Retention recommendation engine
- ✅ FastAPI backend
- ✅ React frontend dashboard
- ✅ Real-time prediction system
- ✅ Clean and responsive UI

# ⚙️ Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/amrutags288-hub/churnzero-ai-platform.git
```

---

## 2️⃣ Backend Setup

```bash
cd churnzero-ai-platform/Backend
```

Install dependencies:

```bash
pip install fastapi uvicorn pandas numpy scikit-learn xgboost
```

Run backend server:

```bash
uvicorn main:app --reload
```

Backend will run on:

```bash
http://127.0.0.1:8000
```

---

## 3️⃣ Frontend Setup

Open new terminal:

```bash
cd churnzero-ai-platform/Frontend
```

Install dependencies:

```bash
npm install
```

Run frontend:

```bash
npm run dev
```

Frontend will run on:

```bash
http://localhost:5173
```
